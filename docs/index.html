Uh-oh, WebGPU not supported :(

This demo requires the modern WebGPU graphics API to run. Seems like your browser does not support it.

Here are the steps to run it in your browser of choice:

  • Chromium (Chrome, Edge, Opera, etc.)

    • WebGPU has begun shipping to Mac / Windows / ChromeOS since Chrome 113, Edge 113 and Opera 99. Please update your browser to a latest version.

  • Firefox

    • WebGPU WebGPU is enabled by default in Nightly Firefox builds.

  • Safari

    • WebGPU is enabled by default in Safari Technology Preview.
    • In macOS Sequoia betas it is testable via the "Feature Flags" in Safari Settings. See release notes for Safari Technology Preview 185.
    • In the iOS 18 betas and visionOS 2 betas, it is available via Settings -> Apps -> Safari -> Advanced -> Feature Flags -> WebGPU
